# Simple wrapper made from [nstools](https://github.com/skydark/nstools)

# Usage

Run `install.sh` script, then you can import python module

```python
import nstools
print nstools.convert('中文簡繁轉換')
# 中文简繁转换
print nstools.convert('中文简繁转换', 'traditional')
# 中文簡繁轉換
```
